# Week 1: Asynchronous Tasks

**Spring 2026 | PUFY 1263: Emergent Objects**

Since our first class is cancelled, please complete the following tasks on your own before our first meeting in Week 2 (Feb 2).

---

## Required Tasks

### 1. Read the Syllabus
Read the complete course syllabus to understand:
- Course expectations and grading
- Attendance and participation policies
- Community agreements
- Materials and supplies needed
- Schedule and assignment due dates

**Link:** [Course Syllabus](https://github.com/arielchuri/emergentobjects/blob/main/.storage/meta/PUFY_1263_A_CHURI_SP26.md)

### 2. Set Up Your Development Environment
Install and configure your Python development environment following the lab guide.

**What you'll do:**
- Install Python 3.10 or newer
- Install Visual Studio Code
- Install the Python extension for VS Code
- Test your setup with a "Hello, world!" program

**Link:** [Development Environment Setup](https://github.com/arielchuri/emergentobjects/blob/main/labs/dev-environment-setup.md)

**Mark as complete:** This is a lab assignment (complete/incomplete). Make sure you can successfully run a Python program before Week 2.

### 3. Complete Lab Orientation
Complete the Making Center lab orientation to gain access to tools and equipment.

**Link:** [Lab Orientation Assignment](https://github.com/arielchuri/emergentobjects/blob/main/assignments/lab-orientation.md)

**Due:** Week 2 (Feb 2)

### 4. Canvas Site Walkthrough
- Log into the Canvas course site
- Familiarize yourself with the course modules
- Add your Learning Portfolio link to your Canvas bio
- Check announcements and upcoming due dates
- Review the course schedule

### 5. Post Your Introduction
Introduce yourself to the class! Share a photo and brief info about yourself, plus complete one of the fun improv prompts.

**What you'll do:**
- Post a photo of yourself
- Share basic info (name, pronouns, why you're taking the course, etc.)
- Choose and complete ONE improv prompt (Object as Origin Story, Future Self, Two Truths and a Fabrication, or Your Personal Object)
- Respond to at least 2 classmates' posts

**Link:** [Introduction Post Assignment](https://github.com/arielchuri/emergentobjects/blob/main/assignments/intro-post.md)

**Due:** Week 2 (Feb 2)

### 6. Start Your Utopia/Dystopia Assignment
Begin brainstorming your speculative world scenario. This foundational work will guide all your design decisions for the semester.

**Assignment:** Create a utopian or dystopian future scenario and develop a user persona living in that world.

**Links:**
- [Utopia/Dystopia LP Post Assignment](https://github.com/arielchuri/emergentobjects/blob/main/assignments/utopia-dystopia-lp-post.md)
- Not due now but included for reference [Persona LP Post Assignment](https://github.com/arielchuri/emergentobjects/blob/main/assignments/persona-lp-post.md)

**Due:** Week 2 (Feb 2) — Initial draft
