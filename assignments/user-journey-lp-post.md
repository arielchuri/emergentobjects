# User Journey Learning Portfolio Post

**Week:** Assigned Week 7, Due Week 8
**Due:** Mar 30
**Weight:** Part of Persona & User Journey (10%)

## Learning Outcomes

This assignment maps to the following learning outcomes:

- **LO 4 - Discipline Specific:** Apply design research methodologies and analyze user behaviors to develop detailed personas and user journeys.
- **LO 7 - Reflection:** Document their process and reflect on their learning via the Parsons Learning Portfolio.

## Description

Map out the complete user journey showing how your persona discovers, obtains, learns to use, and integrates your emergent object into their daily life within your speculative future scenario.

## Requirements

- Create a visual user journey map that includes:
  - Key stages (awareness, acquisition, first use, regular use)
  - User actions at each stage
  - Emotional states and reactions
  - Touch points and interactions
  - Pain points and delights
  - Timeline or sequence
- Written narrative (400-600 words) describing the journey
- Connection to your specific persona and scenario
- Visual format (diagram, illustrated timeline, or storyboard)

## Deliverables

- User journey map (visual)
- Written narrative (400-600 words)
- Posted to Learning Portfolio with proper documentation
- Export PDF and upload to Canvas

## Tips

- Think chronologically through the entire experience
- Include both positive and negative moments
- Show how your object solves problems or creates value
- Consider the context of your future scenario at each stage
- Include sensory details and specific interactions
- Don't just show product features - show human experience
- Think about what happens before and after the main interaction
- Consider different use contexts and environments
